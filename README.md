## 课程论文（`report/`）

**赤道波动力学的算子代数框架**

从三维层结大气基本方程出发，通过垂直模态分解与赤道 β-平面近似，将赤道波动力学纳入统一的算子代数框架。论文覆盖干波理论（Kelvin 波、混合 Rossby-重力波、赤道 Rossby 波与惯性重力波的色散关系与本征结构）、湿过程对波动的修正，以及微扰分析。采用 CoDL 2025 会议模板排版。
